# Google Images Application with API

Aplicación web que permite a los usuarios buscar imágenes en tiempo real utilizando el motor de búsqueda de Google, impulsado por SerpApi para obtener resultados estructurados JSON.

## Tecnologías utilizadas:

Frontend: React.js, CSS3
Backend: Node.js/Express.js y JS para manejar la API Key de forma segura.
API: SerpApi Google Images Result
