# Google Images Application with API

Aplicación web que permite a los usuarios buscar imágenes en tiempo real utilizando el motor de búsqueda de Google, impulsado por SerpApi para obtener resultados estructurados JSON rápidos y precisos.

# Tecnologías utilizadas:

Frontend: CSS y React
Backend: Node.js/Express y JS para manejar la API Key de forma segura.
API: SerpApi Google Images Result
